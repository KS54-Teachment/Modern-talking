## 📝 Краткое описание проекта
Проект **«Цифровой куратор»** направлен на помощь пользователям в работе с цифровыми сервисами (Госуслуги, электронная почта, онлайн-оплата, запись к врачу) и обеспечение их информационной безопасности. Команда фиксирует обращения, готовит памятки и помогает решать типовые проблемы.
