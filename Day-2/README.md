# Цифровой куратор — Команда 3 (Вредоносное ПО)

Учебные материалы дня «Центр цифровой поддержки». Команда 3, тема — **Вредоносное ПО (Malware)**.
В папке собраны результаты всех 6 этапов задания.

## Состав папки

| Файл | Этап | Описание |
|------|------|----------|
| `Etap1_Otchet_Vredonosnoe_PO.docx` | Этап 1. Анализ угрозы | Отчёт: описание угрозы, признаки, реальный пример (WannaCry), последствия, способы защиты |
| `Etap2_Ofitsialnye_istochniki.xlsx` | Этап 2. Информационная разведка | Таблица из 12 официальных источников: тема, источник, дата, достоверность, описание, польза |
| `Etap3_Rabota_s_II.docx` | Этап 3. Работа с ИИ | Инструкция, памятка, FAQ, список вопросов + проверка ответа ИИ и отчёт «Преимущества и недостатки ИИ» |
| `Etap4_Zhurnal_obrashcheniy.xlsx` | Этап 4. Работа с Excel | Журнал обращений: выпадающие списки, условное форматирование, цветовая маркировка, автоподсчёт |
| `Etap5_Baza_znaniy_FAQ.docx` | Этап 5. База знаний | База знаний цифрового куратора — 22 вопроса-ответа в формате FAQ |
| `Etap6_Keys_virusnoe_zarazhenie.docx` | Этап 6. Практический кейс | Кейс №4 «Подозрение на вирусное заражение»: анализ, план действий, рекомендации |

## Тема команды

**Вредоносное ПО** — программы, созданные для нанесения вреда устройству, сети или данным
(вирусы, черви, трояны, шифровальщики, шпионское и рекламное ПО, майнеры, руткиты).

## Цель дня

1. Эффективно искать информацию.
2. Отличать достоверные источники от недостоверных.
3. Изучить основные угрозы информационной безопасности.
4. Освоить базовые принципы работы с ИИ.
5. Пополнить базу знаний команды.
