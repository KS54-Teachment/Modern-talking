## Как создать Issues вручную
1. Откройте репозиторий → вкладка **Issues** → **New issue**.
2. Введите название и описание.
3. Справа назначьте **Assignee**, **Labels** (приоритет), **Milestone**/срок.
4. Для статусов удобно использовать **Projects (board)**: колонки To do / In progress / Done.
