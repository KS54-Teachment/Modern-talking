# 🖥️ Modern Talking — Центр цифровой поддержки

## 📌 Название проекта

**Modern Talking — Центр цифровой поддержки пользователей.**
Командный учебный проект по организации совместной работы через Git и GitHub.

## 🎯 Цель проекта

Научиться работать командой в едином репозитории: вести ветки, делать пул-реквесты, разрешать конфликты, вести задачи через Issues и собрать базу знаний и материалы по цифровой поддержке пользователей.

## 🛠️ Используемые технологии

* **Git и GitHub** — контроль версий, ветки, Pull Request, Issues
* **Markdown** — документация и база знаний
* **HTML / CSS** — страница проекта на GitHub Pages
* **Python 3** — скрипт проверки надёжности пароля
* **Bash** — вспомогательные скрипты
* **MS Office** (Word, Excel, PowerPoint) — отчёты, журналы, презентация
