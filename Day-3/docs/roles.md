# Распределение ролей в команде

## Команда «Modern Talking»

### 🧭 Лидер команды — Комиссаров Илья
- Координация работы команды и сроков.
- Ведение README и структуры репозитория.
- Финальная сборка материалов к защите.
- Ветка: `feature-lead`

### 🛡️ Специалист по ИБ — Чернов Матвей
- Анализ цифровых угроз (вредоносное ПО).
- Подготовка инструкций по информационной безопасности.
- Проверка надёжности паролей и доступов.
- Ветка: `feature-security`

### 🎧 Консультант — Яноха Денис
- Работа с обращениями пользователей.
- Подготовка FAQ и базы знаний.
- Сценарии консультаций.
- Ветка: `feature-consultant`

### 🛠️ Технический специалист — Госани Айдар
- Настройка репозитория и веток, помощь команде по Git.
- Python-скрипты, ведение журнала обращений.
- Ветка: `feature-tech`

### 📝 Документатор — Чирков Костя
- Ведение документации и оформление материалов.
- Логотип и презентация к защите.
- Ветка: `feature-documentation`

## Матрица ответственности (RACI, кратко)

| Задача | Ответственный |
|--------|---------------|
| README и структура | Лидер команды |
| Инструкции по ИБ | Специалист по ИБ |
| База знаний / FAQ | Консультант |
| Журнал обращений / тех. настройка | Технический специалист |
| Документация и оформление | Документатор |
| Защита проекта | Вся команда |
