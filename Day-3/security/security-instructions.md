# Инструкция по информационной безопасности

Тема команды: **Вредоносное ПО**. Краткая памятка для пользователей.

## Базовые правила

1. Установите антивирус и регулярно обновляйте ОС и программы.
2. Не открывайте подозрительные вложения и ссылки.
3. Скачивайте ПО только из официальных источников.
4. Используйте надёжные пароли и двухфакторную аутентификацию.
5. Делайте резервные копии важных данных.
6. Не используйте один пароль для разных сервисов.
7. Блокируйте экран, когда отходите от компьютера.

## Признаки заражения

- Резкое замедление и перегрев.
- Всплывающая реклама и неизвестные программы.
- Отключённый антивирус, недоступные файлы.

## Действия при заражении

1. Отключите интернет.
2. Не вводите пароли.
3. Полная проверка антивирусом.
4. Смена паролей с чистого устройства.
5. При необходимости — переустановка системы.
