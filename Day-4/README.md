## 🧰 Использованные инструменты

- **Oracle VirtualBox 7.2** — гипервизор для виртуальной машины
- **Ubuntu Linux** — гостевая операционная система
- **Терминал (bash)** — команды: `lsb_release -a`, `hostnamectl`, `hostname -I`, `free -h`, `df -h`, `ping`, `ip route`, `mkdir`, `touch`, `cat`, `tree`
- **Guest Additions** — интеграция гостевой ОС с хостом (общие папки, буфер обмена)
- **Git / GitHub** — командная работа с репозиторием
