## 📌 О проекте

«DIGITAL SUPPORT» — настольное приложение цифрового куратора, которое помогает
специалистам Центра цифровой поддержки консультировать граждан, обеспечивать их
цифровую безопасность и вести учёт обращений.

Приложение объединяет **три инструмента**:

1. 🔐 **Генератор надёжных паролей** — криптостойкие пароли на основе модуля `secrets`.
2. 🧪 **Анализатор надёжности пароля** — оценка по 6 критериям безопасности.
3. 🗂️ **Журнал обращений граждан** — регистрация и хранение обращений в JSON.

## 🐞 Исправленная ошибка

В исходной версии приложение падало с ошибкой:

```
_tkinter.TclError: invalid command name "170"
```

**Причина:** в классах виджетов `GButton` ширина/высота сохранялись в атрибут
`self._w` (`self._w, self._h = width, height`). Но `_w` — **служебный атрибут
tkinter**, в котором хранится внутреннее имя виджета (Tcl-путь, например `.170`).
Перезапись `self._w = 170` ломала вызов `self.delete("all")`, который обращался к
несуществующей команде `"170"`.

**Решение:** переименовали поля в `self._cw` и `self._ch`, не трогая служебный
`self._w`. Подробности — в `reports/Итоговый_отчёт.docx` и `project/README.md`.
