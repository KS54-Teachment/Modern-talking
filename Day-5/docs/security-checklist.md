# 🔒 Чек-листы информационной безопасности

Команда **Modern Talking** · Методические рекомендации для граждан

> Ответственный: Чернов Матвей (специалист по ИБ)

---

## ✅ Чек-лист 1. Надёжный пароль

- [ ] Длина не менее 12 символов
- [ ] Есть строчные и прописные буквы
- [ ] Есть цифры и спецсимволы
- [ ] Нет словарных слов, имён, дат рождения
- [ ] Уникален для каждого сервиса
- [ ] Хранится в менеджере паролей, а не на бумажке

## ✅ Чек-лист 2. Защита аккаунтов

- [ ] Включена двухфакторная аутентификация (2FA)
- [ ] Привязаны актуальные почта и телефон для восстановления
- [ ] Проверены активные сессии и устройства
- [ ] Отключён автовход на чужих компьютерах
- [ ] Не сообщаются коды из SMS посторонним

## ✅ Чек-лист 3. Защита от фишинга

- [ ] Проверяется адрес отправителя письма
- [ ] Ссылки проверяются наведением курсора до клика
- [ ] Не открываются вложения от неизвестных отправителей
- [ ] Сайт проверен на https и правильный домен
- [ ] Пароль никогда не вводится по ссылке из письма

## ✅ Чек-лист 4. Безопасность устройства

- [ ] ОБНОВЛЕНы ОС и программы
- [ ] Работает антивирус
- [ ] Включён экран блокировки с PIN/паролем
- [ ] Программы ставятся только из официальных источников
- [ ] Сделаны резервные копии (правило 3-2-1)

## ✅ Чек-лист 5. Защита финансов

- [ ] Оплата только на сайтах с https
- [ ] Отдельная карта для онлайн-покупок с лимитом
- [ ] Подключены SMS-уведомления об операциях
- [ ] CVV и коды не сообщаются никому

## 🚨 Памятка: «Что делать при инциденте»

1. Не паниковать, прекратить опасные действия.
2. Сменить пароли на важных сервисах.
3. Заблокировать карту в банке при финансовой угрозе.
4. Сообщить в службу поддержки сервиса и/или в полицию.
5. Зафиксировать обращение в журнале Центра.
