# 📖 Руководство пользователя и защита проекта

Команда **Modern Talking** · Приложение «DIGITAL SUPPORT»

---

## 1. Установка и запуск

1. Установите **Python 3.8+** с [python.org](https://python.org) (при установке на Windows отметьте «Add Python to PATH»).
2. Откройте папку `project`.
3. Запустите команду:
   ```bash
   python digital_support.py
   ```
4. tkinter входит в стандартную библиотеку Python — дополнительные пакеты не нужны.

## 2. Генератор паролей

- Укажите длину (4–64, рекомендуем 16).
- Включите нужные наборы символов переключателями.
- Нажмите «СГЕНЕРИРОВАТЬ», затем «КОПИРОВАТЬ».

## 3. Проверка надёжности пароля

- Введите пароль — оценка обновляется автоматически.
- Оценка ведётся по 6 критериям, результат — от 0 до 100 баллов.

## 4. Журнал обращений

- Заполните ФИО, тему и описание, нажмите «ДОБАВИТЬ В ЖУРНАЛ».
- Обращения сохраняются в `requests.json`.
- Для удаления выберите запись и нажмите «УДАЛИТЬ».

# 🛡️ Задание 6. Защита проекта — ответы на вопросы

**1. Какую проблему решает ваш проект?**
Многие граждане испытывают трудности с цифровыми сервисами и часто становятся жертвами мошенников из-за слабых паролей. Наш проект помогает цифровому куратору быстро создавать надёжные пароли, оценивать их надёжность и вести учёт обращений граждан — всё в одном простом окне.

**2. Какие технологии использовались?**
Python 3, библиотека tkinter (графический интерфейс), модуль `secrets` (криптостойкая генерация), `json` (хранение данных), `datetime`. Для командной работы — GitHub.

**3. Какие сложности возникли?**
Основная — ошибка `_tkinter.TclError: invalid command name "170"`. Она возникала потому, что в классе кнопки `GButton` ширина хранилась в атрибуте `self._w` — а этот атрибут уже используется самим tkinter для хранения внутреннего имени виджета. Мы нашли причину и переименовали поля в `self._cw`/`self._ch`.

**4. Что можно улучшить?**
Добавить шифрование журнала, экспорт в Excel/PDF, поиск и фильтры по обращениям, интеграцию с базой знаний, проверку паролей по базам утечек, тёмную/светлую темы.

**5. Где проект может применяться на практике?**
В МФЦ и центрах цифровой поддержки, библиотеках, школах и вузах, службах поддержки организаций — везде, где помогают людям с цифровыми сервисами и безопасностью.
