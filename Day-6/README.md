## 🎯 Назначение

Приложение автоматизирует рутинные задачи цифрового куратора и объединяет все 5 заданий
Дня 6 в едином графическом интерфейсе:

| Раздел | Задание | Что делает |
|--------|---------|------------|
| 🏠 Главная | Задание 1 | Приветствие, имя куратора, текущая дата, статистика |
| 🤖 Консультант | Задание 2 | Выбор проблемы → решение и рекомендации |
| 🔐 Пароли | Задание 3 | Генератор паролей + анимированная проверка надёжности |
| 🗂 Обращения | Задание 4 | База обращений: добавить / поиск / удалить (JSON) |
| 📄 Отчёт | Задание 5 | Формирование и сохранение `report.txt` |
| 🛡 Кибербезопасность | Проект 6 | Справочник с поиском |

Командный мини-проект построен на базе «Базы обращений» (Проект 4) и расширен
генератором паролей, FAQ-помощником и справочником по кибербезопасности.

## 🚀 Инструкция запуска

### Быстрый запуск (Windows)
Дважды щёлкните по файлу **`run.bat`** — он сам установит зависимости и запустит приложение.

### Ручной запуск (любая ОС)
Требуется **Python 3.10+**.

```bash
# 1. Установить зависимость
pip install customtkinter

# 2. Запустить
python app.py
```

> macOS / Linux: можно использовать `bash run.sh`.
> На Linux дополнительно может понадобиться пакет `python3-tk`.

## 📁 Структура проекта

```
modern_talking_gui/
├─ app.py            # главное окно, заставка, боковое меню, навигация
├─ views.py          # экраны всех разделов (задания 1-5 + проект 6)
├─ widgets.py        # карточки, кнопки, анимированные виджеты, toast
├─ theme.py          # цветовая палитра и шрифты (стиль massgrave.dev)
├─ logic.py          # вся бизнес-логика (без GUI, легко тестируется)
├─ requirements.txt  # зависимости
├─ run.bat           # запуск для Windows
├─ run.sh            # запуск для macOS / Linux
└─ README.md         # этот файл
```
Файлы данных (`appeals.json`, `report.txt`, `settings.json`) создаются автоматически при работе.

## 🖥 Примеры работы

- **Главная** — приветствие «Добрый день, Илья!», дата, карточки со статистикой обращений.
- **Консультант** — выбираете «Забыл пароль» → получаете решение и 3 рекомендации.
- **Пароли** — слайдер длины, чекбоксы наборов символов, кнопка «Копировать», живой индикатор надёжности.
- **Обращения** — форма добавления, живой поиск, удаление, цветные статусы.
- **Отчёт** — кнопка «Сохранить report.txt» формирует файл с названием команды, датой и списком обращений.
