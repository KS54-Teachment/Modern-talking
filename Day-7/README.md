# Центр цифровой поддержки граждан
Проект создан для Дня 7: «Векторная графика и визуализация информации».

## Что внутри
- Логотип команды в SVG и PNG 1000×1000
- Набор из 5 SVG-иконок: Интернет, Безопасность, Госуслуги, Облако, Поддержка
- Инфографика A4 «10 правил безопасной работы в интернете»
- Социальный плакат A3 для Команды 2: «Безопасные пароли»
- Макет FAQ-страницы
- Excel-журнал обращений с диаграммами и визуальный отчет

## Идея
Центр помогает гражданам безопасно пользоваться цифровыми сервисами, защищать персональные данные и получать понятную поддержку.
