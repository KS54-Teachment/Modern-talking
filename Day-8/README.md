\# День 8. Основы 3D-моделирования в Blender



\## Центр цифровой поддержки граждан



\### Описание проекта



В рамках восьмого дня практики была изучена среда 3D-моделирования \*\*Blender\*\* и выполнен комплекс практических заданий по созданию цифровых объектов и визуализации \*\*Центра цифровой поддержки граждан\*\*.



В ходе работы были освоены базовые инструменты моделирования, работа с материалами, освещением и настройкой камеры, а также получены навыки создания простых сцен и подготовки рендеров.



\---



\## Выполненные задания



\### Задание 1. Изучение интерфейса Blender



Изучены основные элементы интерфейса программы:



\* Окно просмотра сцены (Viewport);

\* Панель объектов (Outliner);

\* Система координат;

\* Инструменты перемещения;

\* Инструменты вращения;

\* Инструменты масштабирования;

\* Object Mode и Edit Mode.



\---



\### Задание 2. Базовые объекты



Создана сцена из стандартных примитивов Blender:



\* Куб;

\* Сфера;

\* Цилиндр;

\* Конус;

\* Тор.



Для каждого объекта изменены положение, масштаб и назначен собственный материал.



\*\*Файл проекта:\*\* `basic.blend`



\*\*Рендер:\*\* `basic.png`



\---



\### Задание 3. Создание 3D-логотипа



На основе логотипа команды создана объемная 3D-модель.



Для модели выполнены:



\* экструдирование;

\* назначение материалов;

\* настройка освещения;

\* итоговый рендер.



\*\*Файл проекта:\*\* `logo.blend`



\*\*Рендер:\*\* `logo.png`



\---



\### Задание 4. Рабочее место цифрового куратора



Создана сцена рабочего места специалиста цифровой поддержки.



Сцена включает:



\* рабочий стол;

\* монитор;

\* ноутбук;

\* клавиатуру;

\* телефон;

\* офисное кресло;

\* настольную лампу;

\* элементы организации рабочего пространства.



При создании использовались собственные объекты и готовые ассеты.



\*\*Файл проекта:\*\* `workspace.blend`



\*\*Рендер:\*\* `workspace.png`



\---



\### Задание 5. Материалы и освещение



Для сцены рабочего места были настроены:



\* материалы объектов;

\* источники света;

\* положение камеры;

\* параметры рендера.



Получено качественное изображение, пригодное для презентации проекта.



\---



\### Задание 6. Визуализация центра цифровой поддержки



Разработан концепт современного центра цифровой поддержки граждан.



В проект включены:



\* зона консультирования;

\* рабочие места специалистов;

\* информационный стенд;

\* фирменная цветовая палитра;

\* современный минималистичный дизайн.



При создании использовались собственные модели, готовые ассеты и материалы.



\*\*Файл проекта:\*\* `future\_workspace.blend`



\*\*Рендер:\*\* `future\_workspace.png`
