# Учебные приложения Windows Forms (C#)

Набор из четырёх настольных приложений на C# / .NET 8 (Windows Forms),
выполненных по заданиям 1–5. Все приложения оформлены в едином стиле
(синяя тема + логотип в шапке).

## Состав

| Папка | Приложение | Задание |
|-------|------------|---------|
| `Greeting` | Приветствие | Задание 1 |
| `Calculator` | Калькулятор | Задание 2 |
| `PasswordGen` | Генератор паролей | Задание 3 |
| `DigitalSupport` | Центр цифровой поддержки | Задания 4 |
| `CyberSecurityCenter` | Центр обращений | Задания 5 |
---

## Описание проектов

### Greeting — «Приветствие» (Задание 1)
Простое приложение с заголовком, текстовым полем и кнопкой.
- Пользователь вводит имя и нажимает кнопку **«Поздороваться»**.
- В результате выводится приветствие: `Здравствуйте, {имя}!`.
- Есть проверка на пустое поле.

### Calculator — «Калькулятор» (Задание 2)
Калькулятор с основными арифметическими действиями.
- Сложение, вычитание, умножение, деление.
- Кнопка **C** — очистка формы.
- Проверка деления на ноль (сообщение «Делить на ноль нельзя!»).
- Защита от некорректного ввода (сообщение «Ошибка»).

### PasswordGen — «Генератор паролей» (Задание 3)
Генератор надёжных паролей.
- Генерация пароля по заданным параметрам.
- Выбор длины пароля (от 4 до 64 символов).
- Наборы символов: заглавные буквы, строчные буквы, цифры, спецсимволы.
- Опция «без похожих символов» (l, I, O, 0, 1 и т.п.).
- Кнопка **«Копировать»** — копирование результата в буфер обмена.
- Оценка надёжности пароля (полоса прогресса + цветная метка).

### DigitalSupport — «Центр цифровой поддержки» (Задания 4)
Журнал обращений пользователей.
- Поля: ФИО (TextBox), описание проблемы (TextBox), статус обращения (ComboBox), дата обращения (DateTimePicker).
- Таблица обращений (DataGridView).
- Кнопки: **Добавить**, **Изменить**, **Удалить** запись.
- Проверка заполнения полей.
- **Работа с файлами (Задание 5):** все обращения сохраняются в текстовый файл `data.txt`,
  автоматически загружаются при запуске и восстанавливаются после перезапуска программы.

### CyberSecurity — «Центр кибербезопаснсти» (Задания 5)

- Приложение состоит из трёх основных разделов.
Справочник угроз
Каталог распространённых киберугроз.
Список угроз (фишинг, вредоносное ПО, слабые пароли, социальная инженерия и др.).
При выборе угрозы отображается её описание и признаки.
- Рекомендации
Практические советы по защите.
Рекомендации по безопасности (надёжные пароли, двухфакторная аутентификация, обновления и т.п.).
Помогают снизить риски из справочника угроз.
- Тест безопасности
Короткий проверочный тест на знание основ кибербезопасности.
Вопросы с вариантами ответов.
Подсчёт результата и итоговая оценка уровня знаний.

## Инструкция запуска

### Важно: снять веб-метку (если архив был скачан)
Windows помечает скачанные из интернета файлы, и сборка может выдать ошибку **MSB3821**
(«файл находится в Интернете или ограниченной зоне»). Перед сборкой снимите метку:

- **Способ 1 (рекомендуется):** ПКМ по скачанному `.zip` → **Свойства** →
  поставить галочку **«Разблокировать» (Unblock)** → **ОК** → и только потом распаковать.
- **Способ 2:** если уже распаковано, открыть PowerShell в папке проектов и выполнить:
  ```powershell
  Get-ChildItem -Path . -Recurse | Unblock-File
  ```

### Вариант A — через Visual Studio
1. Открыть файл решения нужного приложения, например `WinFormsApp4/WinFormsApp4.sln`.
2. Нажать **F5** (запуск с отладкой) или **Ctrl+F5** (запуск без отладки).

### Вариант B — через командную строку (.NET CLI)
Перейти в папку конкретного проекта и выполнить:
```bash
cd CyberSecurityCenter
dotnet run
```
(Аналогично для `PasswordGen`, `Calculator`, `Greeting`, `DigitalSupport`)
